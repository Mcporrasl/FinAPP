## 2024-05-24 - [Array Processing Optimizations]\n**Learning:** Found multiple instances of chained `.filter().reduce()` operations inside React render cycles (e.g., in `HomeTab.tsx` and `DreamsTab.tsx`), which created an O(M*N) performance overhead.\n**Action:** Replaced chained array methods with single-pass loops wrapped in `useMemo` to prevent unnecessary calculations on every render, improving general app responsiveness.
